What is EndGate?

EndGate is a game framework that is used to build powerful HTML5 games. Powered by TypeScript, large gaming projects become easier to create and maintain. Built with the intent of making complex HTML5 games simple and fun to create EndGate provides developers with a multitude of API's to control everything from keyboard input, mouse input, audio management, graphics handling, camera control, collision detection and many other useful features that are key to building a quality game.
